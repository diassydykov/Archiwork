# Archiwork

Генератор архитектурных проектов для частных домов и коммерческих зданий (ТРЦ).

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

## Возможности

- Вход и регистрация пользователя
- 3 языка: қазақша, русский, English
- Темы: светлая, тёмная, для слабовидящих
- Выбор типа здания: частное / коммерческое
- Форма деталей проекта

## Деплой на Vercel

1. Подключите репозиторий в Vercel
2. Добавьте переменные: `LEONARDO_API_KEY`, `STABILITY_API_KEY`
3. Деплой запустится автоматически

## Инфраструктура Alem.ai

| Сервис | Назначение | Статус в проекте |
|--------|------------|------------------|
| **Qwen 3.6** | Улучшение промптов перед генерацией | Подключено |
| **MinIO S3** | Хранение сгенерированных изображений | Подключено |
| **Deepseek OCR** | Распознавание планов/документов | Готово к подключению |
| **PostgreSQL** | Пользователи и проекты в БД | Позже (сейчас localStorage) |
| **Langflow** | Визуальные AI-пайплайны | Опционально, вне приложения |

### Переменные окружения

Скопируйте `.env.example` → `.env.local`. **Не коммитьте `keys.txt` и `.env.local`.**

## AI-провайдеры (изображения)

| Провайдер | Где ключ | Роль |
|-----------|----------|------|
| Leonardo AI | [leonardo.ai/api-access](https://app.leonardo.ai/api-access) | Основной |
| Stability AI | [platform.stability.ai/account/keys](https://platform.stability.ai/account/keys) | Запасной |

Если Leonardo не сработал — автоматически используется Stability.

```bash
cp .env.example .env.local
```

## Что не нужно для старта

- **Bing / Azure OpenAI** — платная подписка Azure, для MVP не нужно
- **Ideogram** — проблема входа в браузере, публичного API нет
- **Adobe Firefly** — API только для корпоративных клиентов Adobe

Для Archiwork достаточно **Leonardo + Stability**.
